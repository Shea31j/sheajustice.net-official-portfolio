# sheajustice.net — Personal Portfolio

Live site: [sheajustice.net](https://sheajustice.net)

A single-page personal portfolio built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools — just clean, well-commented code.

## Features

- Responsive design with dark / light mode toggle
- Scroll-reveal animations
- Animated progress bars (Codecademy CS path)
- Bat Cave — a playable Flappy Bird-style canvas game with a 3-layer parallax sewer background
- Contact form (Formspree-ready)
- Pan-African flag with tooltip

## Built with

- HTML5, CSS3, JavaScript (ES6+)
- HTML5 Canvas API (game)
- Google Fonts (DM Serif Display, DM Sans, DM Mono)
- Hosted on Netlify
